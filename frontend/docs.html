🧠STACK PERMANENTE
Codex Spark — cerveau, gros code, archi, debug sale, review finale.
Z.ai — assistant code, exécution, refactors mécaniques, worker parallèle.
ElevenLabs — voix, TTS, tout ce qui touche au vocal.
FAL.ai — médias rapides : photos, images, vidéos, prototypage créatif.
RunPod — GPU, jobs lourds, pipelines, infra contrôlée.
OpenRouter — centre API pour les besoins spécifiques et les modèles ponctuels.
⏳OPPORTUNITÉS TEMPORAIRES
Gemini — crédits à cramer intelligemment, surtout image et tests ciblés.
Grok / xAI — veille X/Twitter temps réel tant qu'il reste du crédit.
✂️À COUPER
MiniMax — intéressant mais trop redondant si pas d'avantage massif.
Claude abonnement — plus assez cohérent, surtout avec le délire anti 3rd harness.
🎯DOCTRINE DE DISPATCH
Penser fort / coder critique → Codex Spark
Coding rentable / délégation d'exécution → Z.ai
Voix → ElevenLabs
Média rapide → FAL.ai
GPU ou gros pipeline → RunPod
Modèle spécifique ponctuel → OpenRouter
Crédits ou bonus temporaires → Gemini / xAI
