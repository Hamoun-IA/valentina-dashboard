# Valentina Dashboard 🎯

Dashboard de monitoring pour Hermes Agent.

## Features
- Vue d'ensemble de l'activité
- Suivi des coûts par provider
- Statistiques d'utilisation des modèles
- Monitoring des tâches cron
- Analytics médias (images, audio, vidéo)

## Stack
- **Backend:** FastAPI
- **Frontend:** HTML/CSS/JS + Chart.js
- **Data:** Hermes Agent session DB + logs
